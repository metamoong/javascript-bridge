## 기능 목록

1. 입력

- [ ] 생성할 다리 길이
  - [ ] 3~20이 아니면 예외 처리
- [ ] 플레이어가 이동할 칸 입력
  - [ ] U or D 가 아니면 예외 처리
- [ ] 게임 재시작/종료 여부 입력
  - [ ] R or Q 가 아니면 예외 처리

2. 출력

- [ ] 게임 시작 문구
- [ ] 게임 종료 문구
- [ ] 다리 건너기 결과 출력
- [ ] 게임 재시작 여부 확인 문구

3. 기능

- [ ] 플레이어가 건널 수 있는 칸을 선택했는지 판별
- [ ] 게임 플레이 상태 저장
- [ ] 게임 성공 여부 판별
- [ ] 게임 재시작
- [ ] 게임 종료
- [ ] 게임 시도 횟수 저장
